# Raw QM9 data

Place the original, **unmodified** QM9 download here (e.g. `dsgdb9nsd.xyz.tar.bz2`
and the extracted `.xyz` files, plus the authors' `readme.txt`).

Source: Figshare collection 978904 — Ramakrishnan, Dral, Rupp & von Lilienfeld,
*Quantum chemistry structures and properties of 134 kilo molecules*,
Scientific Data 1, 140022 (2014).

> Files in this folder are git-ignored on purpose. Raw data is treated as
> read-only and is never edited in place.
